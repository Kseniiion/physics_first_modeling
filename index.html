Оптимизация катушки

Расчёт длины намотки, при которой магнитная индукция на оси в центре катушки максимальна.

Оптимум

График B = f(l)

Физическая модель

Внешний диаметр провода с изоляцией: a = d + 2δ. Радиус центра витков j-го слоя:

r_j = D/2 + a/2 + j a

Поле каждого витка в центре катушки:

B_i = μ₀ I r_i² / (2(r_i² + x_i²)^(3/2))

Индуктивность выводится двумя оценками: формулой длинного соленоида и формулой Уиллера для короткой воздушной катушки.
